Minimum dependencies on higher level packages

Core dependencies:

    python = ">=3.8,<3.11"

    pip install vanilla_option_pricers

    pip install --upgrade vanilla_option_pricers



Supported Option types (passed as string):

    CALL = 'C'
    PUT = 'P'
    INVERSE_CALL = 'IC'
    INVERSE_PUT = 'IP'
